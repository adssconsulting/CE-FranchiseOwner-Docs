# CE FranchiseOwner Docs

Documentation site for [CE FranchiseOwner](https://github.com/adssconsulting/CE-FranchiseOwner),
served at <https://adssconsulting.github.io/CE-FranchiseOwner-Docs>.

## How it's published

- Source: every `.md` file under `docs/` plus `mkdocs.yml`.
- Build: GitHub Actions runs `mkdocs gh-deploy --force` on every push
  to `main`. The workflow lives in `.github/workflows/deploy.yml`.
- Theme: MkDocs Material, indigo palette.

## How content is updated

The content under `docs/` is regenerated by running the **Docs
Engine** tab in the CE FranchiseOwner Admin window. That tab copies
a prompt to the clipboard; pasting it into Claude Code triggers
Claude to read the current state of the source repository and
overwrite the `.md` files in this repository with fresh content.

## Local preview

```bash
pip install mkdocs-material
mkdocs serve
# Open http://127.0.0.1:8000
```
