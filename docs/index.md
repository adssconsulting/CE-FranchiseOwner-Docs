# CE FranchiseOwner

Three tools. One platform. Built for Commission Express franchise owners.

## Lead Engine

**Automated targeted outreach to active real estate agents — on your schedule, under your control.**

Every day the app finds real estate agents whose deals just went Active Under Contract in South Carolina.

[Read the Lead Engine guide →](marketing/feature-guides.md#lead-engine)

## Call Tracker

**A smart call logger that keeps your previous client relationships alive.**

Tracks every call made to previous or inactive clients — who you called, what they said, and when to follow up.

[Read the Call Tracker guide →](marketing/feature-guides.md#call-tracker)

## Pre-Underwriting / Gate Check

**Know if a deal is worth your time before you spend a minute on it.**

Every application email that arrives in your inbox gets automatically scored against 12 key criteria.

[Read the Gate Check guide →](marketing/feature-guides.md#pre-underwriting-gate-check)

## How this site is maintained

This site is regenerated by the Docs Engine tab inside the app's Admin window. The operator clicks **Copy Prompt**, pastes the prompt into Claude Code in VS Code with the source repository open, and Claude reads the current state of the code and overwrites the markdown files in the [CE-FranchiseOwner-Docs](https://github.com/adssconsulting/CE-FranchiseOwner-Docs) repository. A GitHub Actions workflow auto-publishes the rebuilt site to GitHub Pages within two minutes of every push to `main`.

Each successful regeneration appends a line to `logs/doc_run.log` in the source repo; the Docs Engine tab reads that file to set its status banner colour (green when up to date, amber while a run is in progress, red on failure, grey when never run).
