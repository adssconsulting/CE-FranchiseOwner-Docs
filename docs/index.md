# CE FranchiseOwner

CE FranchiseOwner is the Windows desktop application built for
**Commission Express of South Carolina**. It is two products under one
roof, both used daily by a franchise owner:

## Lead Engine

A semi-automated lead pipeline for real-estate agent outreach:

- **Scrape** PropStream for *Active Under Contract* listings across
  every South Carolina county, using saved searches stored in the app.
- **Match** the listing agents on those properties against a master
  roster of licensed SC real-estate agents (rapidfuzz fuzzy match,
  human verification workflow).
- **Email** the matched agents a personalised drip sequence (10 stops)
  through Rackspace SMTP, escalating the cadence by contact count, and
  attaching each agent's open-listing table to the relevant template.
- **Track** opens, sends, unsubscribes (polled twice daily from the
  IMAP inbox), and cold-call follow-ups.

## Pre-Underwriting

A scoring engine for the *other side* of the Commission Express
business — buying agent commission receivables before settlement:

- **Receive** completed application emails into a dedicated IMAP
  mailbox, parsed automatically into encrypted DB rows (Fernet
  AES-128 for every PII field).
- **Classify and scan** the application's attached PDFs (purchase
  contract, MLS sheet, pre-qual letter, compensation form, driver's
  license, misc) using `pdfplumber` for local text extraction and
  Claude Haiku 4.5 for two targeted reads:
  - **Q4** — open contingencies on the purchase contract
  - **Q8** — mortgage pre-approval / proof of funds
- **Score** every application against a 12-question gate check with
  knockout logic, returning a verdict the franchise owner uses to
  decide whether to advance funds.
